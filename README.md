# CHIP-8

A CHIP-8 emulator for the Java Virtual Machine written in Ceylon.

### Key Mapping

The CHIP-8 machine has a 16-key hexadecimal keyboard (`0` - `F`).

These are mapped to the keys on a typical QWERTY keyboard:

```
1 2 3 4
 Q W E R
  A S D F
   Z X C V
```
